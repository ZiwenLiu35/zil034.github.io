Large Model training Optimization

This project focuses on optimizing the training of Clip models within Artificial Intelligence and Graphics Computing (AIGC) systems by manipulating key parameters. The primary goal is to retain high throughput while minimizing GPU memory costs. The research involved the exploration of various techniques, resulting in a 35.4% reduction in GPU memory costs through the implementation of ZeRO. This optimization was achieved while simultaneously increasing throughput by 2.09 units of data samples. Key strategies include manual adjustments to batch size and enabling communication overlap.
